HTML5

HTML5 es la quinta revisión del lenguaje HTML, que se utiliza para estructurar y presentar contenido en la web. Introduce nuevas etiquetas y atributos que facilitan la creación de páginas web más semánticas y accesibles.


Podemos hacer uso del pluggin incluyído en VSC "emmet" para facilitarnos el trabajo:
Por ejemplo, si queremos crear una estructura básica de HTML5, podemos escribir "! + tabulador"? o "html:5 + enter" y Emmet generará automáticamente el código necesario para una página HTML5 básica.
Aquí tenemos un enlace a la página oficial de Emmet para más información, o inluso un enlace al pdf resumen de atajos de Emmet.

La estructura básica de un doc html incluye:

  • <!DOCTYPE html>: Declara el tipo de documento y la versión de HTML.
  • <html>: Elemento raíz que contiene todo el contenido de la página.

  • <head>: Contiene metadatos, enlaces a hojas de estilo, scripts, y el título de la página.

    1. <meta charset="UTF-8">: Define la codificación de caracteres.
    2. <meta name="viewport" content="width=device-width, initial-scale=1.0">: Hace que la página sea responsiva en dispositivos móviles.
    3. <title>: Define el título de la página que aparece en la pestaña del navegador.
    4. <link rel="stylesheet" href="styles.css">: Enlaza una hoja de estilo externa para dar estilo a la página.
    5. <script src="script.js"></script>: Enlaza un archivo JavaScript externo para agregar funcionalidad a la página.
    6. Si quisiéramos añadir metas:
      <meta name="keywords" content="palabra clave1, palabra clave2, palabra clave3">

  • <body>: Contiene el contenido visible de la página, como texto, imágenes, enlaces, etc.

    Podemos estructurar el contenido del documento con una serie de etiquetas que tienen sentido semántico po sí mismas:


    1. <header>: Define el encabezado de una página o sección.
    2. <nav>: Define un bloque de navegación con enlaces.
    3. <main>: Define el contenido principal de la página.
    4. <section>: Define una sección genérica de contenido.
    5. <article>: Define un contenido independiente y autocontenido.
    6. <aside>: Define contenido relacionado pero separado del contenido principal, como barras laterales.
    7. <footer>: Define el pie de página de una página o sección.

    NOTA:
    Tanto header como footer pueden repetirse en los diferenter articles o sections repartidos en el main

    Dentro del body podemos encontrar múltiples etiquetas como:

    1. <h1> a <h6>: Encabezados de diferentes niveles.
    2. <p>: Párrafos de texto.
    3. <a href="url">: Enlaces a otras páginas o recursos.
    4. <img src="imagen.jpg" alt="descripción">: Imágenes con una descripción alternativa.
    5. <div>: Contenedores genéricos para agrupar elementos.
    6. <span>: Contenedores en línea para agrupar texto o elementos.
    7. <ul> y <ol>: Listas no ordenadas y ordenadas.
    8. <li>: Elementos de lista dentro de listas.
    9. <header>: Encabezado de una sección o página.
    10. <footer>: Pie de página de una sección o página.
    11. <section>: Sección genérica de contenido.
    12. <article>: Contenido independiente y autocontenido.
    13. <nav>: Navegación de la página.
    14. <form>: Formularios para la entrada de datos del usuario.
    15. <input>: Campos de entrada dentro de formularios.
    16. <button>: Botones interactivos.

    NOTA:
    Existen infinidad de etiquetas HTML5, pero estas son algunas de las más comunes y utilizadas para estructurar y presentar contenido en la web.
    Es importante utilizar las etiquetas de manera semántica para mejorar la accesibilidad y el SEO de las páginas web. Para más información, puedes consultar la documentación oficial de HTML5.

CSS3

CSS3 es la tercera versión del lenguaje CSS, que se utiliza para diseñar y dar estilo a las páginas web. Incluye nuevas características como transiciones, animaciones, y flexbox, que permiten crear diseños más complejos y atractivos.


Podemos incluir CSS3 en nuestros documentos HTML de tres maneras diferentes:

  1. CSS Externo: Se crea un archivo separado con extensión .css (por ejemplo, styles.css) y se enlaza al documento HTML utilizando la etiqueta <link> dentro del <head>. Ejemplo:
    <link rel="stylesheet" href="styles.css">
  2. CSS Interno: Se incluye dentro de una etiqueta <style> en el <head> del documento HTML. Ejemplo:
    <style> body { background-color: lightblue; } </style>
  3. CSS Inline: Se aplica directamente a un elemento HTML utilizando el atributo style. Ejemplo:
    <h1 style="color: blue;">Hola Mundo</h1>

Es recomendable utilizar CSS externo para mantener el código organizado y facilitar el mantenimiento.


Siempre empezaremos el doc css con:
/* CSS Document */

Un ejemplo muy común al empezar a trabajar con CSS es el siguiente reset CSS, que nos ayuda a eliminar los márgenes y paddings por defecto de los navegadores, y a establecer un box-sizing consistente:

*{ margin: 0; padding: 0; box-sizing: border-box; }

  • margin: Establece el margen externo de todos los elementos a 0, eliminando cualquier espacio adicional alrededor de ellos.
  • padding: Establece el relleno interno de todos los elementos a 0, eliminando cualquier espacio adicional dentro de ellos.
  • box-sizing: border-box: Cambia el modelo de caja para que el ancho y alto de los elementos incluyan el padding y el borde, lo que facilita el control del tamaño total de los elementos.
  • Este reset CSS es útil para crear un punto de partida consistente para el diseño de páginas web, ya que los diferentes navegadores aplican estilos predeterminados que pueden variar.
  • Al utilizar este reset, podemos asegurarnos de que los elementos se comporten de manera predecible y uniforme en todos los navegadores.
  • Después de aplicar este reset, podemos comenzar a agregar nuestros propios estilos CSS para diseñar la página web según nuestras necesidades.
  • Para más información sobre CSS3, puedes consultar la documentación oficial de CSS3.

Uso de CSS Flexbox

¿Qué es Flexbox?
Flexbox (Flexible Box) es un modelo de diseño de CSS3 pensado para distribuir espacio entre elementos dentro de un contenedor flexible, incluso cuando su tamaño es desconocido o dinámico.

    Permite alinear y distribuir elementos de manera eficiente en una sola dimensión (fila o columna).

    Facilita la creación de diseños responsivos y adaptativos.

    Proporciona un control más preciso sobre la alineación, el orden y el tamaño de los elementos.

    Propiedades principales en el contenedor (Padre)

  • display: flex — Activa el modo flexbox.
  • flex-direction — Dirección (fila o columna) de los elementos ("row", "row-reverse", "column", "column-reverse").
  • flex-wrap — ¿Permite que los elementos se vayan a otra línea? ("nowrap", "wrap", "wrap-reverse").
  • justify-content — Alinea en el eje principal ("flex-start", "center", "flex-end", "space-between", "space-around", "space-evenly").
  • align-items — Alinea en el eje transversal ("flex-start", "center", "flex-end", "stretch", "baseline").
  • align-content — Espacio entre líneas cuando hay varias (solo si wrap).

    Propiedades principales en los ítems (Hijos)

  • order — Orden visual de los elementos.
  • flex-grow — Capacidad para crecer si hay espacio.
  • flex-shrink — Capacidad para reducirse si hay poco espacio.
  • flex-basis — Tamaño base inicial.
  • flex — Shorthand de las anteriores.
  • align-self — Sobrescribe alineación de un hijo concreto.


Ejemplo de código CSS Flexbox:

.container0 {
display: flex;
justify-content: space-around;
align-items: center;
}
.item0 {
background-color: lightcoral;
padding: 20px;
text-align: center;
}
